# Set your API key

FoxxyCode needs a provider API key before the agent can call a model.

## Inside the embedded UI

1. Open the **FoxxyCode** panel (activity bar icon).
2. Click **Settings** (gear icon in the UI or the webview toolbar).
3. Paste your provider API key and save.

## Alternative: environment variable

Set `OPENAI_API_KEY` in your shell or system environment **before** launching VS Code.

## Extension settings

For host, port, binary path, and language, open **Settings → Extensions → FoxxyCode**, or run **FoxxyCode: Settings** from the Command Palette.
