# Welcome to FoxxyCode

The **foxxycode** agent binary is bundled with this extension — no download or install step.

![FoxxyCode activity bar icon](../foxxycode.svg)

## Find the panel

1. Look at the **activity bar on the left** and click the **FoxxyCode** icon (shown above).
2. Or press **Ctrl+Shift+P** (Cmd+Shift+P on macOS), type **FoxxyCode: Open Panel**, and press Enter.

The extension starts `foxxycode http` scoped to your workspace and embeds the full web UI in the sidebar.

---

Click **Next →** to set up your API key.
