# Toolbar shortcuts

The **FoxxyCode** webview title bar (sidebar or editor tab) provides quick actions:

| Button | Action |
| --- | --- |
| **Restart** | Stop and restart the `foxxycode http` process |
| **Reload** | Refresh the embedded UI iframe |
| **Open in Browser** | Open the same UI in your system browser |
| **Open DevTools** | Inspect the webview (useful when the UI goes blank) |
| **Settings** | Jump to **Settings → Extensions → FoxxyCode** |

**Show Logs** is available from the Command Palette (**FoxxyCode: Show Logs**) — stdout/stderr from the foxxycode process.

The extension version is shown next to the view name in the sidebar header.

## Attach a file by dragging it

Drag a file from the **Explorer** or an open **editor tab** onto the composer to insert it as a short `@`-mention. **Hold `Shift` while dragging** — VS Code only lets a workbench drag reach a webview while `Shift` is held.
